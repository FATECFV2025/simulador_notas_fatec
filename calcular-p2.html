Explicação

Como funciona:
N1 = P1 (70%) + AT1 (30%)

MF = (N1 + N2) / 2

Para PASSAR você precisa de MF ≥ 6,0
Para estar em EXAME/P3 você precisa de 2,0 ≤ MF < 6,0
A P3 só existe quando a MF fica entre 2,0 e 5,9
